The directory has all source and output files.
use 'cat setvar' to see the contents of the file. Run the command given in the file on prompt to set environment variable.
Go to apps directory and run 'make' command.
Open a separate terminal and set the environment variable again for this terminal.
Run './server' in 1 terminal. Run './client' in the other terminal.
The client terminal will ask for a filename. Enter a valid filename.
The file will be sent to the server and copied as recv_file in the same directory.
